CHM

Если в настройках книги на книжной полке была включена опция buildChm, то рядом с html-статикой будут созданы файлы my_book_key.hhp и my_book_key.hhс. Для сборки chm нужно только скачать SDK и запустить компилятор

hhc my_book_key.hhp

Для генерации chm, лучше использовать темы и макеты без левого меню, как, например blueprint_chm.
