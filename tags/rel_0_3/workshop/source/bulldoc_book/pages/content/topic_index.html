Предметный указатель строится на основе ключевых слов присвоенных страницам.

Ключевые слова назначаются страницам с помощью служебного тэга:

<cls:keywords data='установка, настройка' />

Страница указателя добавляется в любое место оглавления, которое удобно. К обычному описанию страницы в этом случае добавляется ее тип: index.

....
appendix:
  type: chapter
  title: Приложения
  topics:
    topic_index.html: {type: index, title: Предметный указатель}
    similar.html:  Аналоги
    roadmap.html:  Планы по развитию

Для оптимизации скорости работы система кэширует файл с индексом. Поэтому если Вы добавляли новые ключевые слова страницам и хотите увидеть их в предметном указателе, сбросьте кэш, нажав "построить index" в правом верхнем углу веб-интерфейса.
