HTML

Экспорт в статический html выполняется из командной строки

bulldoc bulldoc

Для экспорта книги в один большой html-файл, нужно явно указать этот режим, добавьте такую строку в запись файла книжной полки (config/bookshelf.yml):

outputMode: html_single

Для генерации монолитного файла, тема должна содержать отдельные файлы макета без левого меню, и специальными выражениями для ссылок, см., как это сделано в теме blue_print.

CHM

Если в настройках книги на книжной полке был задан режим outputMode: chm, то рядом с html-статикой будут созданы файлы my_book_key.hhp, my_book_key.hhс и my_book_key.hhk. Для сборки chm нужно только скачать SDK и запустить компилятор

hhc my_book_key.hhp

Для генерации chm, необходимо, чтобы тема имела соответствующие макеты макеты без левого меню, как, например в теме blueprint.

Будьте внимательны, насколько нам известно, формат CHM, не имеет поддержки utf-8. Пользуйтесь 8-битными кодировками для создания CHM.
