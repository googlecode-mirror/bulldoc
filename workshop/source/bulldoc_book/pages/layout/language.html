Язык интерфейса можно задать, как для системы целиком, так и отдельно для каждой книги.

Язык интерфейса системы задается в /local_config.inc.php

  $CONFIG['language']='rus';
  $CONFIG['locale']='cp1251';

Следует отличать локаль от языка. Язык определяет текст, а локаль значения кодировок в заголовках, в работе с текстом, сортировке и т.д. Поэтому вполне жизнеспособной может быть настройка с английским языком интерфейса, но с русским текстом, которому необходима русская локаль. Поддерживаются языки: русский-windows1251 (rus), русский-utf8 (utf8_rus) и английский (eng)

Если Вы хотите назначить UTF-8 в качестве рабочего языка для всей системы, не забудьте сконвертировать в utf-8 файл книжной полки (config/bookshelf.yml).

Можно также задать язык для отдельных книжек. Для этого в параметрах книги на книжной полке надо задать язык:

language: rus
locale: cp1251 #этот параметр можно не указывать, 
               #будет использована локаль для языка по умолчанию

Для работы с utf-8:

language: utf8_rus

Будьте внимательны, насколько нам известно, формат CHM, не имеет поддержки utf-8.
