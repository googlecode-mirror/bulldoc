Настройки

Настройки оформления, которые влияют на систему целиком находятся в двух файлах:

config/docgen_config.ini.php
messages/docgen.msg.ini.php

В файле docgen_config.ini.php задается путь к шаблону оформления списка книг, который можно видеть зайдя в корень проекта, и к файлу помощи для утилиты построения статического html. Также здесь задается название класса, который пропускает через себя содержимое страницы, осуществляя в частности подсветку кода. Интерфейс этого класса см. ниже

В файле docgen.msg.ini.php содержатся строковые константы для заголовка страницы обложки и для текста страниц, которые еще не созданы.
