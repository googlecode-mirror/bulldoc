HTML

Экспорт в статический html выполняется из командной строки

bulldoc bulldoc

Для экспорта книги в один большой html-файл, добавьте строку в запись файл книжной полки (config/bookshelf.yml):

singlePageExport: 1

Для генерации монолитного файла, лучше использовать темы и макеты без левого меню, и специальными выражениями для ссылок, например blue_print_single.

CHM

Если в настройках книги на книжной полке была включена опция buildChm, то рядом с html-статикой будут созданы файлы my_book_key.hhp, my_book_key.hhс и my_book_key.hhk. Для сборки chm нужно только скачать SDK и запустить компилятор

hhc my_book_key.hhp

Для генерации chm, лучше использовать темы и макеты без левого меню, как, например blueprint_chm.

Будьте внимательны, насколько нам известно, формат CHM, не имеет поддержки utf-8. Пользуйтесь 8-битными кодировками для создания CHM.
