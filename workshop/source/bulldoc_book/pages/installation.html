Скачать и распаковать

Вам потребуется php 5.2+ (обратите внимание, что для работы из командной строки, нужна консольная версия php. Например, в Linux (K)Ubuntu необходимо поставить пакет php5-cli).

Если Вы собираетесь использовать режим веб-сервера, то у Apache должен быть подключен mod_rewrite. Это делается прописыванием в httpd.conf строки

LoadModule rewrite_module modules/mod_rewrite.so

В Linux (K)Ubuntu модуль можно подключить командой

a2enmod rewrite.load

Ваш виртуальный сервер должен понимать файл .htaccess. Для этого в конфигурационный блок для Вашего сайта (либо он внутри httpd.conf, либо отдельным файлом, например в (K)Ubuntu это /etc/apache2/sites-available) надо вписать строку: 

LoadModule rewrite_module modules/mod_rewrite.so

Скачайте программу с сайта. Распакуйте архив.

Настроить

После распаковки система готова к работе. Позаботьтесь, чтобы у программы были права на запись в папку workshop и папку cache. Для работы из командной строки под Unix, дайте права на исполнение файлу /bulldoc

В настоящий момент не предполагается использовать Bulldoc на production-сервере, т.к. возможны проблемы с безопасностью.
