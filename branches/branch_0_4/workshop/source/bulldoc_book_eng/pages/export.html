HTML

Export to html is performed from command line

bulldoc bulldoc

If you want to export to the single html file, you should indigate this by the line in book's settings on the bookshelf  (config/bookshelf.yml):

outputMode: html_single

CHM

If you set outputMode: chm, then my_book_key.hhp, my_book_key.hhс and my_book_key.hhk files will be created near html files. U just need to run chm compiler (Download SDK)

hhc my_book_key.hhp
