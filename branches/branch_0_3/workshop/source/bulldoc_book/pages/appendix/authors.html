Многие друзья дали мне ценный совет или просто поддержали идею этого проекта, за что им большое спасибо.

В программе используется подсветка синтаксиса GeSHi, распространяется по лицензии GPL. Если Вы хотите использовать BullDoc в проекте с несвободной лицензией, вам придется заменить модуль подсветки.

Для работы с YAML-файлами использовался SPYC, распространяется по MIT лицензии.

Для темы по умолчанию был активно использован CSS-фреймворк Blueprint. Он распространяется по MIT лицензии.

Для работы меню в теме по умолчанию использованы буллеты с сайта PHP
